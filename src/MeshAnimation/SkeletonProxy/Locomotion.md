# Locomotion

任务：角色动画的实时控制

## Motion Graph / Motion Matching

优势：
1. 基于准备好的数据库，即可以与角度进行实时控制
2. 

缺点：
1. 依赖海量数据
2. 最近邻搜索成本高
3. 内存占用大
4. 对不同地形泛化性差

目标：
1. 多：更复杂的场景、更多地动作
2. 快：实时交互、快速地数据准备、快速响应
3. 好：动作质量高、动作合理、动作合物理、符合用户控制
4. 省：更少的内存、更少的数据、更少的人工参与、更少的预处理
